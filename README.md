# Personal Portfolio Website

## Overview

The Personal Portfolio Website is a responsive frontend web project developed using HTML, CSS, and JavaScript. It is designed to showcase personal skills, projects, resume, and contact information in a professional and interactive format.

This project was created to improve frontend development skills and strengthen understanding of responsive layouts, animations, smooth navigation, and UI/UX design.

---

## Features

✅ Responsive Portfolio Layout
✅ About Section
✅ Skills Showcase
✅ Projects Section
✅ Resume Download Button
✅ Contact Information Section
✅ Smooth Scroll Navigation
✅ Hover Effects and Animations
✅ Mobile-Friendly Design
✅ Interactive User Interface

## Functionalities

### 👤 About Section

Displays personal introduction and career objective.

### 🛠️ Skills Section

Showcases technical skills and technologies.

### 📂 Projects Section

Displays completed projects with descriptions and preview images.

### 📄 Resume Download

Users can download resume directly from the website.

### 📞 Contact Section

Provides contact details and social media links.

### ✨ Animations & Hover Effects

Interactive transitions improve the user experience.

### 📱 Responsive Design

Portfolio adapts smoothly for mobile, tablet, and desktop screens.

### 🌐 Smooth Navigation

Smooth scrolling between sections using navigation links.
